== Description ==

Prevent users from accessing and downloading files in the wp-contents, ex: .txt, .doc, .pdf, .zip, ... except images.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/block-file-access` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
